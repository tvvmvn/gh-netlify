프론트엔드란?

웹개발

인터넷 또는 인트라넷용 웹사이트를 만드는 것 또는 웹사이트를 만드는 것과 관련된 분야

서버와 클라이언트

서버

리소스 또는 서비스의 제공자

클라이언트

리소스 또는 서비스를 요청하는 사람(컴퓨터)

프론트엔드와 백엔드

프론트 엔드

애플리케이션의 보여지는 부분 (presentation layer)

백 엔드

애플리케이션의 데이터 처리 부분(data access)

프론트엔드 개발언어

HTML (Hyper Text Markup Language)

웹페이지의 데이터를 표시한다

CSS (Cascading Style Sheet)

웹페이지의 디자인을 처리한다

JavaScript

HTML과 CSS를 조작하여 동적인 페이지를 구현하는데 사용된다

백엔드 개발언어

PHP

세계적으로 가장 많이 사용되는 서버 구축용 언어
예) 워드프레스, 페이스북 등

Java

한국에서 가장 많이 사용되는 언어. 전자정부프레임워크에서 사용된다

Python

서버구축, 빅데이터, 머신러닝 등 다양한 분야에서 사용되는 언어

NodeJS

JavaScript를 사용하여 서버를 구축할 수 있게 만든 언어
